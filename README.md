## Featured Projects / 代表项目

### Data Science / Machine Learning
- **[wids-2026](https://github.com/xyjk0511/wids-2026)**
  WiDS Datathon 2026 clinical prediction pipeline with gradient boosting ensembles, calibration, and leaderboard score 0.97089.

- **[vesuvius-surface-detection](https://github.com/xyjk0511/vesuvius-surface-detection)**
  3D segmentation pipeline for the Kaggle Vesuvius Challenge, evolving from custom U-Net baselines to nnUNetv2 / ResEnc L with leaderboard-focused postprocessing.

- **[modma-eeg-depression](https://github.com/xyjk0511/modma-eeg-depression)**
  EEG-based depression classification on MODMA with ERP/P300 feature engineering, subject-level validation, and statistically significant results.

- **[tdbrain-eeg-classification](https://github.com/xyjk0511/tdbrain-eeg-classification)**
  EEG classification pipeline on TDBRAIN for psychiatric disorder prediction with cross-validation, feature engineering, and AUC 0.798.

### Applied AI / Engineering
- **[huaweiocr](https://github.com/xyjk0511/huaweiocr)**
  End-to-end OCR pipeline for device-label extraction with detection, barcode decoding, OCR, and structured JSONL export.

- **[codex-review](https://github.com/xyjk0511/codex-review)**
  AI-assisted code review workflow with multi-agent validation, issue detection, and optional auto-fix.

### Signal Processing / Specialized Projects
- **[tomatis-audio-processor](https://github.com/xyjk0511/tomatis-audio-processor)**
  Python implementation of a Tomatis-style audio processor with dynamic gated filtering and signal-processing-based calibration.
